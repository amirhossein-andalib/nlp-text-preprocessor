# NLP Text Preprocessor — Persian & English

A bilingual NLP preprocessing pipeline supporting both Persian and English text.

## Features

### Persian
- Text Normalization (Hazm)
- Tokenization
- Stopword Removal
- Lemmatization & Stemming
- Diacritics Removal

### English
- Lowercasing
- Punctuation & Number Removal
- Tokenization (NLTK)
- Stopword Removal
- Lemmatization (WordNet)

## Installation
pip install hazm nltk

## Usage
python main.py

## Technologies
- Python 3.x
- [Hazm](https://github.com/roshan-research/hazm) — Persian NLP
- [NLTK](https://www.nltk.org/) — English NLP
